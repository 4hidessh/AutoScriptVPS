__**Text Format Guide**__

Code: ``*ITALIC TEXT*``
Output: *TEXT*

Code: ``**BOLD TEXT**``
Output: **TEXT**

Code: ``***BOLD+ITALIC TEXT***``
Output: ***TEXT***

Code: ``__UNDERLINED TEXT__``
Output: __TEXT__

Code: ``__*UNDERLINED ITALIC TEXT*__``
Output: __*TEXT*__

Code: ``__**UNDERLINED BOLD TEXT**__``
Output: __**TEXT**__

Code: ``__***UNDERLINED BOLD TEXT***__``
Output: __***TEXT***__

Code: ``~~SPIKETHROUGH TEXT~~``
Output: ~~TEXT~~
``(You can mix with other code here)``

Code: Add 2 backticks before the TEXT and also at the end
Output: `TEXT`

Code: Add 3 backticks before the TEXT and also at the END
Output: ```TEXT```

Code: ``||SPOILER TEXT||``
Output: ||TEXT||
``You can mix with other code here``
